### & (按位与)

```
a = 2, b = 3

a & b = 2 

// 解释：都为1的时候得1，出现0的时候得0
// 2的二进制为'10', 3的二进制为'11'，结果为'10'，即2
```

- 使用 `&` 判断一个数的奇偶

###  ^ (按位异或)

- 利用^来完成比较两个数是否相等
```
1 ^ 1 = 0

1 ^ 非1数 ！=0

所以同一个数^同一个数等于0，否则不等于0
```

- 使用^来完成值交换
```
a = 1

b = 2

a ^= b

b ^= a

a ^= b

结果a=2，b=1
```
